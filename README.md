# Project Admin Multikart

## Quick Start

Clone repo
```
git clone [https://github.com/thanhit23/multikart.git](https://github.com/thanhit23/book-store.git)
cd book-store
npm install
```

Set the environment variables:

```
cp .env.example .env
```
